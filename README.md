Overview
Rubric Checker is an interactive website that helps students track their assignment progress in real time. As users complete each requirement on the rubric, the system automatically checks off the tasks, providing live feedback and ensuring that no step is missed before submission.
This project was originally designed in Figma and then implemented as a simple, responsive website.

Features
Interactive checklist that updates as tasks are completed
Live progress feedback that shows completion percentage or visual ticks
Rubric-based task list aligned with assignment requirements
Simple and clean layout inspired by writing center and peer connection support
Responsive design that works on both desktop and mobile devices

Technologies Used
HTML for structure
CSS for layout and design
JavaScript for interactivity and live feedback
